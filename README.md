## How to run the project

### Clone repository
```
git clone https://github.com/Dnzldotexe/wasims-dev.git
```

### Navigate to project
```
cd wasims-dev/
```

### Setup development environment
```
chmod u+x dev-setup.sh
./dev-env-wasims.sh
```

# How to stop the project

### Stop npm
Press keys `CTRL + C`

### Stop sail container
```
sail down
```

### Run it again
```
sail up -d
sail npm run dev
```
